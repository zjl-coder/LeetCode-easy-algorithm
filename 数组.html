删除排序数组中的重复项，返回新长度

  • 借用新数组，遍历数组，判断是否在新数组中，在舍去，不在就推入新数组，计算新数组长度
  • 指针法：先将数组排序，然后 判断当前则和i + 1是否相等，是则跳过累加，否则标记 + 1

旋转数组 ， 数组中的元素向右移动 k 个位置 ,有可能k比length大

  • 遍历，借用新数组， newindex = (oldindex + k) % length
  • 循环k次，每次都将第一个元素临时存起来，从数组尾部开始遍历（newindex = (oldindex+1) % length）然后将临时元素放入index = 1的地方
  • 拆分，将倒数（k % length）个元素拆出来 分成两部分，然后将第一部分拼接在第二部分的后面
  • 数组翻转，整体翻过来，然后将前面（k % length）个元素翻一下，后面 length - (k % length) 个元素翻一下

存在重复元素

  • 双指针：先排序，再遍历判断 i 和 i+1 是否相等

两个数组的交集 II

sort 排序的时间复杂度是 nlogn 空间是 logn
  • 哈希表，记录第一个数组的元素出现的次数，遍历第二个数组，如果存在哈希表中，则对应元素次数-1,并且将改元素放入新数组
  • 双指针法，两个数组排序，然后两个指针分别指向数组的头部，对比元素，相等则放入新数组，不同，则移动较小的数字的指针

移动零

  • 双指针法： first 指向第一个0，second 找第一个非0元素（这样first和second之间都是0）交换位置，然后 first second 同时 移动，如果second碰到0则只移动second

两数之和，数组 nums 和一个整数目标值 target，找出和为目标值的那 两个 整数

  • 暴力循环，将元素与剩下的元素相加判断是否满足目标
  • 借用新数组，target - arr[i] 得到的值放入新数组，在放之间先检查是否在数组中，如果已存在，则说明当前值下标 和 新数组所对应的下标就是我们要的

有效的数独

  • 迭代（循环）：行列，子数独一起遍历， boxIndex = Math.floor(row / 3) * 3 + Math.floor(col / 3)
    分别存入三个二维数组 行列和子数独 行的一维下标是row 然后每次存进去判断是否已存在同一维下标的元素有值，列的一维下标是 col，子数独的一维下标是 boxindex
